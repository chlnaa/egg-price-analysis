# 🥚 달걀 소매가격 변동 구조 분석

### 원가 전이 메커니즘과 외부충격 구간 식별을 통한 매입 의사결정 지원 | 2016~2025

---

## 💡 핵심 인사이트

- **사료비(feed_ppi)** 가 달걀 가격에 가장 큰 영향
  — RF 변수 중요도 feed_ppi 단독 약 36%, feed 계열 합산 약 94% / SHAP 기여값 +600원대

- **전력(electricity_ppi)** 은 elec 계열 합산 약 6%로 영향력 미미
  — OLS 음(-)의 계수는 다중공선성 왜곡으로 인과 해석 불가

- **전체 R²=0.188은 두 레짐 혼합에 의한 왜곡**
  — 평상시(n=94) R²=0.808 / AI충격기(n=23) R²=0.121로 원가-가격 연동 사실상 단절
  — "추가 변수 필요"가 아닌 **"레짐을 구분한 모델이 필요"**

## 📌 프로젝트 개요

달걀은 국민 필수 식품으로 2016~2025년 AI 대란 및 원자재 급등 등으로 가격 급변이 반복됐습니다.

**핵심 질문**

> "사료비 및 에너지 원가가 오르면 달걀 소매가격은 얼마나, 어떤 구조로 변화하는가?  
> 그리고 외부충격이 발생할 때 이 구조는 어떻게 바뀌는가?"

| 가설                                                        | 결과          | 핵심 근거                                                    |

## 📝 한계점 및 발전 방향

**한계점**

- OLS 잔차 자기상관 존재(Durbin-Watson=0.254) — 시계열 모델(ARIMA) 필요
- test 구간(2024~2025) 월별 등락은 원가 변수만으로 설명 불가  
  — 수급과 계절 변수 추가 필요
- 계절성 진폭(±220원, 7.4%)은 외부충격 대비 미미 — 독립 피처 채택 시 효과 제한적
- 공급측 변수(산란계 사육수, 살처분 규모) 미확보 — H3 수급 모니터링 제안의 검증 한계

**발전 방향**

- SQLite → PostgreSQL 마이그레이션 (진행 예정)
- Streamlit 대시보드 구축 — 지역별 가격 지도 시각화 포함
- KAPE 산란계 통계·농림부 살처분 이력 데이터 추가 확보
- H4 검증: 원가 누적 압력과 가격 바닥 구조 변화
  - 2021년 이후 평상시 가격 평균 6,597원으로 구조적 상향 관찰됨
  - 예비 분석 R²=0.783, p<0.001 — 레짐 전환 1회로 추가 기간 확보 후 재검증 필요
  - 현재 원가 수준 유지 시 평상시 소매가 하한선 약 6,200원으로 추정
- 시계열 모델(ARIMA, LSTM) 적용 검토
