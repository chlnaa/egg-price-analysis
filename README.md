# 🥚 鶏卵小売価格の変動構造分析

### コスト転嫁メカニズムと外部ショック期間の識別による仕入れ意思決定支援 | 2016~2025

---

## 💡 核心インサイト

「平常時と外部ショック時で原価–価格の連動構造がどう変わるのか」を定量的に確認しました。

- **飼料費(feed_ppi)** が鶏卵価格に最も大きな影響
  - RF変数重要度：feed_ppi単独 約36%、feed系列合算 約94% / SHAP寄与値 +600ウォン台

- **電力費(electricity_ppi)**はelec系列合算 約6%で影響軽微
  - OLSの負の係数は多重共線性による歪みのため因果解釈は不可

- **全体R²の0.188は2つのレジームが混在することによる歪み**
  - サンプル数94件の平常時のR²では0.798、サンプル数23件のAIショック期のR²では0.020まで低下しており、コスト–価格連動が事実上断絶
  - 単純な価格予測にとどまらず、**「平常時と外部ショック時でコスト転嫁構造がどのように変わるのか」** を定量的に確認しました。

---

## 📌 プロジェクト概要

鶏卵は生活必需食品であり、2016〜2025年にかけて鳥インフルエンザの大規模発生や原材料急騰などにより価格変動が繰り返されました。

**核心となる問い**

> 飼料費やエネルギーコストが上昇すると、鶏卵小売価格にどれほど影響するのか？  
> 外部ショック発生時、コストの連動構造はどのように変わるのか？

`year_month`を用いたJOINやLAG処理の際に日付型の変換が都度必要になったため、  
TIMESTAMP型を明確にサポートするPostgreSQLへ移行しました。  
環境構築はDocker Composeを採用し、`docker compose up -d`で再現できる構成にしました。

## 🔬 仮説と検証結果

| 가설                                                   | 결과         | 핵심 근거                                                                   |
| ------------------------------------------------------ | ------------ | --------------------------------------------------------------------------- |
| H1: 飼料費はラグなしで鶏卵小売価格に即時転嫁される     | 条件付き採択 | lag0の時`r=0.462`が最高値、ラグ増加で単調減少を確認。ただし転嫁強度は限定的 |
| H2: 電力費より飼料費の影響力が大きい                   | 採択         | RFのfeed系列重要度94% vs elec系列6%、SHAPも同方向　                         |
| H3: 外部ショック期にはコスト以外の要因が価格を支配する | 採択         | AIショック期のfeed相関 −0.141 vs 平常時 0.893、R² 0.798 -> 0.020へ崩壊      |

**核心的解釈**
全期間を単一モデルで扱うと説明力が低く見えますが、レジームを分離すると構造が明確になります。

- 平常時：コスト変数だけで価格変動の約80%を説明可能
- AIショック期：コスト–価格連動が事実上崩壊し、コスト以外の要因が価格を支配していたと推察される

> コスト変数自体が弱いのではなく、外部ショック時に価格形成メカニズムそのものが転換する

## 🤖 モデル性能比較

| モデル                 | RMSE          | MAE           | 備考             |
| ---------------------- | ------------- | ------------- | ---------------- |
| OLS 重回帰分析         | 286ウォン     | 226ウォン     | ベースライン     |
| Random Forest          | 274ウォン     | 229ウォン     | コスト特徴量のみ |
| XGBoost                | 344ウォン     | 281ウォン     | 過学習傾向を確認 |
| RF + ai_dummy + regime | **272ウォン** | **228ウォン** | 最終モデル       |

> train: 2016.04〜2023.12 / test: 2024.01〜2025.12（時間順分割）

**最終モデル選択の根拠**

RF+ai_dummyのRMSE改善幅はOLS比14ウォンと数値上の差は小さいです。  
しかしサンプル数94件の平常時のfeed_ppi相関が0.893であるのに対し、AIショック期のサンプル数23件では−0.141へと逆転する構造的断絶が確認されており、  
この断絶をモデルに明示的に反映することが単純な性能指標の改善より重要と判断しました。  
ai_dummyとregime変数のRF重要度合算がコスト変数の合算を上回ったことも、この判断を裏付けています。

## 📝 限界と今後の課題

**限界点**

- OLS残差に自己相関が確認され、Durbin-Watsonは0.254であった — 時系列モデル未適用
- 採卵鶏飼養数や殺処分規模などの供給側変数の組み込みは今回の分析範囲外とし、  
  コスト–価格構造の定量化に集中した。そのため7,000ウォン超の急騰局面5ヶ月は全モデルで捉えられなかった
- 2017年の殺虫剤卵パニックなど需要側ショックに対する別途変数を未反映

今回はコスト–価格構造の定量化に分析範囲を絞りました。
供給側データの組み込みと時系列モデルの適用は次のステップの課題としています。

**今後の発展方向**

- 月別AI発生件数を供給ショックの代理変数として組み込み、
  AIショック期の急騰局面の説明力向上を図る
- 採卵鶏飼養数の分期データの月次補間処理による供給側変数の追加
- ARIMA、LSTM時系列モデルの適用と比較

## 📌 プロジェクトの意義

コスト変数が平常時には価格変動の約80%を説明するものの、外部ショック時にはその説明力が崩壊するという事実を、  
レジーム分離とSHAPによって定量化しました。  
これにより「どの局面でコストモデルを信頼でき、  
どの局面では別の指標に切り替えるべきか」という具体的な判断基準を提示しています。
